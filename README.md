# 🚀 DevScout  

## 🛠️ Project Overview

**DevScout** is a project-sharing platform for developers! It allows users to showcase their projects, connect with other developers, provide ratings on projects, and search for developers with specific skills. This project serves as an excellent foundation for learning Django, and introduces concepts like user authentication, search functionality, and building APIs with Django REST Framework (DRF).

### Key Features
- 🌍 **Share Projects**: Post your work for others to see and interact with.
- 💬 **Message Other Developers**: Send direct messages to other developers.
- ⭐ **Rate Others' Work**: Provide ratings to encourage and improve the developer community.
- 🔍 **Search Developers**: Use advanced search to find developers by skills and projects.

## 🖥️ Installation

1. **Clone the Repository**  
   ```bash
   git clone https://github.com/navin-shankar-v/DevScout.git
   ```

2. **Create and Activate a Virtual Environment**  
   ```bash
   pip install virtualenv
   virtualenv envname
   envname\scripts\activate  # For Windows
   source envname/bin/activate  # For MacOS/Linux
   ```

3. **Navigate to the Project Directory**  
   ```bash
   cd Django-Course
   ```

4. **Install Dependencies**  
   ```bash
   pip install -r requirements.txt
   ```

5. **Run the Application**  
   ```bash
   python manage.py runserver
   ```

## 📚 Course Overview

This project covers a wide range of Django concepts, ideal for beginners looking to deepen their understanding. Key topics include:

- 🏗️ **Basic Django Overview**: Gain a foundational understanding of Django.
- 📊 **Database Design and Models**: Learn how to design databases and work with Django models.
- 🎨 **Static Files Management**: Serve and manage static files like CSS, images, and JavaScript.
- 🔐 **User Registration & Authentication**: Implement user sign-ups, logins, and profile management.
- 🔍 **Search Functionality**: Add search functionality to find projects and developers.
- 📄 **Pagination**: Divide content into manageable pages.
- 🔗 **Building an API with Django REST Framework (DRF)**: Create APIs to extend functionality and support frontend frameworks or mobile apps.
- ☁️ **Deployment**: Learn how to deploy Django applications.
